# Student Management System

A production-quality, full-stack Student Management System built with **Java 21 + Spring Boot** (backend) and **React** (frontend).

## Overview

This application provides a comprehensive platform to manage student records, including enrollment, course assignments, grading, and reporting. It is designed to be clean, modular, scalable, and maintainable — suitable for portfolios and production environments.

---

## Tech Stack

| Layer      | Technology                          |
|------------|-------------------------------------|
| Backend    | Java 21, Spring Boot, Spring Data JPA |
| Database   | MySQL                               |
| Frontend   | React, Axios                        |
| Build Tool | Maven (backend), npm (frontend)     |
| Docs       | Postman, Markdown                   |
